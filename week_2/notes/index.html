  • Primitive Data Types: string, symbol, number, boolean, undefined, null
  • Operand: operator applies an operation to a value
  • special operator called typeof for finding out the type of a value
  • const is used when the variable will not be reassigned to another value
  • let is used if the variable might be reassigned later in the program
  • block scoped: their value only exists inside the block they are declared in
  • global scope: Any variable declared outside of a block
  • local scope: variables defined inside a block using the let or const will only be available inside that block and not be accessible outside of that block
  • Constant and variable names are case sensitive
  • Camel case starts with a lowercase letter and then each new word is capitalized
  • Underscore separates each new word with an underscore
  • A method is an action that a primitive data type or object can perform
  • Template literals: special types of string that were introduced in ES6,backtick character, ` , to deliminate the string, expression can be inserted inside a string and the result will be displayed
  • Symbols are the only primitives that don't have a literal form. The only way to create them is to use the Symbol() function
  • Numbers can beintegers(whole numbers, such as 3) orfloating point numbers(often referred to as just 'decimals' or 'floats', such as 3.14159)
  • Hexadecimal or "hex" numbers are often used for color codes on the Web.
  • If a number starts with a zero, followed by the letter o , then it is considered to be in octal (base 8) notation
  • If a number starts with a zero, followed by the letter b then it is considered to be in binary (base 2) notation
  • Non-primitive data types, such as arrays, functions and objects, all have a type of 'object'
  • Boolean values are either true or false
  • Logical operators can be used to check if compound statements are true or false

    • Arrays, Logic, and Loops

  • An array is an ordered list of values
  • Destructuringan array is the concept of taking values out of an array and presenting them as individual values
  • The length property can be used as part of the index to find the last item in an array
  • pop() - to remove the last item from an array
  • shift()- to remove the first item from an array
  • push() - appends a new value to the end of an array
  • unshift() - appends a new item to the beginning of an array
  • concat() - used to merge an array with one or more arrays
  • join() - used to turn array into a string that comprises all items in the array, separated by commas
  • slice() - creates a subarray, chopping a slice of the original array
  • splice() - removes items from an array and inserts new items in their place
  • reverse() - permanently reverses the order of the array
  • sort() - sort the order of the array
  • multidimensional array - array of arrays
  • Sets: ata structure that represents a collection of unique values, so it cannot include any duplicate values
  • Maps are new in ES6 and are ordered lists of key-value pairs
  • We can use an if and else statement to control the flow of code
  • The switch statement can be used instead of multiple if and else statements
  • A while loop and do ... while loop can be used to repeat a block of code while a condition is still true
  • A for loop works in a similar way to a while loop, but has a different syntax
  • A for-of loop can be used to iterate over an array
  • Sets and maps are enumerable, so can also be looped over using a for-of loop

    • Functions

  • To define a function literal we can use a function declaration
  • Another way of defining a function literal is to create a function expression
  • A function can also be declared using the constructor Function()
  • If you have assigned a function to a variable, you need to place parentheses after the variable to invoke it as a function
  • All functions return a value, which can be specified using the return statement, which comes after the return keyword
  • any parameters a function needs are set when the function isdefined
  • When a function isinvoked, it is provided with arguments
  • Arrow functions can be identified by the 'arrow' symbol, => that gives them their name
  • Arrow functions are always anonymous, so if you want to refer to them, you must assign them to a variable
  • callback - A function that is passed as an argument to another
  • forEach() - loop through the array and invoke a callback function using each value as an argument
  • map() - it returns a new array that replaces each value with the return value of the callback function
  • reduce() - cumulatively combines each result to return just a single value
  • filter() - returns a new array that only contains items from the original array that return true when passed to the callback

    • Arrow functions have a number of advantages over other ways of declaring functions

  • They are much less verbose than normal function declarations
  • Single parameters don't need putting into parentheses
  • The body of the function doesn't need placing inside a block if it's only one line
  • The return keyword isn't required if the return statement is the only statement in the body of the function
  • They don't bind their own value of this to the function
