Object methods, "this"

  • A function that is the property of an object is called its method
  • sayHi() : same as "sayHi: function()"
  • To access the object, a method can use the this keyword
  • The value of this is the object “before dot”, the one used to call the method
  • "this" can be used in any function
  • The value of this is defined at run-time
  • When a function is declared, it may use this, but that this has no value until the function is called

    • This in Javascript

  • When you create a new instance of an object with the new keyword, this refers to the instance
  • this within any method refers to the object itself
  • Since this refers to the object, you can use methods to get the instance of an object
  • this in an arrow function is always the same as this around it

    • Objects

  • An object literal is an object that is created directly in the language by wrapping all its properties and methods in curly braces {}
  • object literals: distingushing feature of javascript, quickly create objects, no need for defining class
  • mutable: properties or methods can be changed or removed
  • short ES6 way - const ironMan = { name, realName };
  • 'in' operator can be used to check whether an object has a particular property
  • hasOwnProperty() - to check whether an object has a property that is its own
  • for in - loop through all of an object’s properties and methods
  • Object.values() - returns an array of all the object's value
  • Object.entries() - returns an array of key-value pairs
  • New properties and methods can be added to objects at any time in a program
  • never rely on the properties being in a certain order
  • change the value of an object’s properties at any time using assignment
  • Any property can be removed from an object using the delete operator
  • objects are assigned byreference, if a variable is assigned to an object that already exists, it will simply point to the exact same space in memory
  • object literal pattern - creating an object literal that serves as the namespace, then adding any values as properties of that object, and any functions as methods
  • JavaScript Object Notation, or JSON
  • JSON is a string representation of the object literal notation
  • The parse() method takes a string of data in JSON format and returns a JavaScript object
  • The stringify() method does the opposite, taking a JavaScript object and returning a string of JSON data
  • All the properties and methods of the Math object are immutable and unable to be changed
  • Math object has eight properties that represent a mix of commonly used math constants. Note that they are all named in capital letters
  • Math.PI // The ratio of the circumference and diameter of a circle
  • Math.SQRT2 // The square root of 2
  • Math.SQRT1_2 // The reciprocal of the square root of 2
  • Math.E // Euler's constant
  • Math.LN2 // The natural logarithm of 2
  • Math.LN10 // The natural logarithm of 10
  • Math.LOG2E // Log base 2 of Euler's constant
  • Math.LOG10E // Log base 10 of Euler's constant
  • Math.abs() method returns the absolute value of a number
  • Math.ceil() method will round a numberupto the next integer, or remain the same if it is already an integer
  • Math.floor() method will round a numberdownto the next integer, or remain the same if it is already an integer
  • Math.round() method will round a number to thenearestinteger
  • Math.trunc() method that returns the integer-part of a number – that is, it gets truncated at the decimal point
  • Math.exp() method will raise a number to the power of Euler’s constant
  • Math.pow() method will raise any number (the first argument) to the power of another number (the second argument)
  • Math.sqrt() method returns the positive square root of a number
  • Math.cbrt() method was introduced in ES6, which returns the cube root of numbers
  • Math.hypot() method was also introduced in ES6. It returns the square root of the sum of the squares of all its arguments
  • Math.log() method returns the natural logarithm of a number
  • Math.max() method returns the maximum number from its arguments
  • Math.min() method unsurprisingly returns the minimum number from the given arguments
  • Math.sin() returns the sine of an angle
  • Math.cos() returns the cosine of an angle
  • Math.tan() returns the tangent of an angle
  • Math.asin() returns the arcsine of a number
  • Math.acos() returns the arccosine of a number
  • Math.atan() returns the arctangent of a number
  • The Math.random() method is used to create random numbers, which can be very useful when writing programs

    • Document Object Model - DOM

  • an HTML document as a network of connected nodes that form a tree-like structure
  • everything on the webpage is a node
  • All nodes have a numerical code to signify what type they are
  • Node lists are array-like objects, but they are not arrays
  • getElementById() - reference to the element with a unique id attribute
  • getElementsByTagName() - return a live node list of all the elements with the tag name that is provided as an argument
  • getElementsByClassName() - live node list of all elements that have the class name that is supplied as an argument
  • document.querySelector() - use CSS notation to find thefirstelement in the document that matches that matches a CSS selector provided as an argument
  • document.querySelectorAll() - uses CSS notation but returns a node list ofallthe elements in the document that match the CSS query selector
  • jQuery is a popular JavaScript framework that makes it very easy to find elements on a page using a CSS-style syntax
  • childNodes property is a list of all the nodes that are children of the node concerned
  • children property only returns anyelementnodes that are children of that node
  • firstChild property returns the first child of a node
  • lastChild property returns the last child of a node
  • parentNode property returns the parent node of an element
  • nextSibling property returns the next adjacent node of the same parent
  • previousSibling property returns the previous adjacent node
  • createElement() method that takes a tag name as a parameter and returns that element
  • CSS can be updated - CamelCase notation
  • style property - see what CSS styles have been set on an element

    • Events

  • click event occurs when a user clicks with the mouse, presses the Enter key, or taps the screen, making it a very useful all-round event covering many types of interaction
  • addEventListener() method is called on a node object, the node to which the event listener is being applied
  • document.body.addEventListener('click',doSomething);
  • screenX and screenY properties show the number of pixels from the left and top of the screen respectively where the event took place
  • clientX and clientY properties show the number of pixels from the left and top of the client that is being used
  • pageX and pageY properties show the number of pixels from the left and top, respectively, where the event took place in the document
  • You should be very cautious of attaching both a click and doubleclick event to the same element
  • mousedown and mouseup events
  • hree events that occur when a user presses a key are: keydown , keypress and keyup
  • modifier keys such asShift,Ctrl,Altandmeta(Cmdon Mac)
  • touchstart event occurs when a user initially touches the surface
  • touchend event occurs when a user stops touching the surface
  • touchmove event occurs after a user has touched the screen then moves around without leaving
  • touchenter event occurs when a user has already started touching the surface, but then passes over the element to which the event listener is attached
  • touchleave event occurs when the user is still touching the surface, but leaves the element to which the event listener is attached
  • touchcancel event occurs when a touch event is interrupted, such as a user’s finger moving outside the document window, or too many fingers being used at once
  • There are no 'swipe' events. These need to be created by using a combination of touchstart , touchmove , and touchleave events that monitor the distance and direction moved from start to finish of a touch event
  • An event listener can be removed using the removeEventListener() method
  • preventDefault() is a method of the event object that can be used inside the callback function to stop the default behavior happening
  • each event object has a property called cancellable that returns false if it cannot be prevented
