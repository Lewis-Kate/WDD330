Stragegic Menu Design

  • most commonly used items located at bottom in easily reached area = happy user
  • creates perception of speed
  • improves engagement rates
  • increase user satisfaction
  • evolving resarch can improve user satisfaction

    • Mobile Friendly Forms

  • simple so user can easily use
  • reduce cognitive load
  • float labels & display inside text field
  • keep forms short/only necessary fields
  • progress indicator to show user what is left
  • enable auto complete
  • credit cards auto advance to next field and space between #'s
  • use appropriate keyboard for information you are asking for
  • anchor tag so user can tap phone to call and doesn't have to copy/paste

    • Mobile Advertising

  • research before advertising
  • $25-$150 billion spenet in 5 yr. period
  • STOP USING POP UPS
  • small/simple ads are effective
  • users come for content
  • video ads = higher conversion rate

    • Mobile First

  • start small & scale up
  • focus on content & core functionality
  • mobile devices replaced every 1-2 yrs.
  • time consuming & expensive
  • best user experience
  • addresses constraints of size/bandwidth
  • future friendly
  • ready for growth
  • responsive retrofitting: make what you have mobile friendly, cheaper & faster, doesn't address touch interaction issue

    • Understanding Users

  • Be There
  • Be USeful
  • Be Quick
  • smooth/seamless interactions
  • Rechability: 75% rely on thumbs, 49% have 1 handed grip
  • bottom of device easiest to reach/even when device is landscape
  • speed matters
  • networks matter: 1.8% users 4G, 24.5% user 3G
  • reduce image usage/SVG or webfonts are lower bandwidth
  • optimize/minify CSS & JS files
