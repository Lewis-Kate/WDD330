Using Fetch

  • basic fetch request:
    fetch('http://example.com/movies.json')
    .then(response => response.json())
    .then(data => console.log(data));
  • Use fetch() to POST JSON-encoded data
  • Files can be uploaded using an HTML input element, FormData() and fetch().
  • headers objects have a guard property
  • Response instances are returned when fetch() promises are resolved
  • there is a Fetch Polyfill available that recreates the functionality for non-supporting browsers
